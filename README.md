# Army Profile Portal

A secure, high-performance profile management system designed for defense personnel.

## Features

-   **Secure Authentication**: Role-based access control with secure login/signup.
-   **Profile Management**: Comprehensive profile viewing and editing capabilities.
-   **Unit Management**: Track and manage unit details efficiently.
-   **Tactical UI**: Custom-built interface with Light, Dark, and Camo themes for different operational environments.
-   **Responsive Design**: Fully optimized for desktop and mobile devices.

## Tech Stack

-   **Frontend**: React, TypeScript, Vite
-   **Styling**: Tailwind CSS, Shadcn UI
-   **State Management**: React Query
-   **Routing**: React Router DOM

## Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Rishiofficial432-432/army-profile-portal.git
    cd army-profile-portal
    ```

2.  **Install dependencies**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
